FBX File Structure
The FBX file is named "Axe_Warrior_Full_Set" and includes the character rig along with all three armor sets.

Armor Set Naming Conventions
The armor sets are prefixed for easy identification:

A1_ → Armor Set 1

A2_ → Armor Set 2

A3_ → Armor Set 3

All armor pieces are grouped under empty GameObjects:

Helmet Objects → Contains all helmets from A1, A2, A3

Body Armor → Contains Body armor

Hand Equipments → Contains all hand armor

Leg Equipments → Contains all leg armor
